# False Memory Word List Worksheet

Name: ________________________________  Class: __________________  Date: __________________

## Learning Focus

Coherence can produce confident false memory.

## Student Prompt

Inspect False Memory Word List Stimulus A. Record one first claim, the evidence you used, your confidence, and what would make you revise.

## Concrete Stimulus Packet

### False Memory Word List Stimulus A

Student-facing stimulus 1: A word list around sleep that omits the word 'sleep'. Before discussing, write what you noticed, what you inferred, and how confident you are.

Teacher cue: Students inspect this exact card first and commit to a claim before hearing anyone else.

### False Memory Word List Stimulus B

Student-facing stimulus 2: A second list around sweet foods that omits 'candy' or 'sugar'. Before discussing, write what you noticed, what you inferred, and how confident you are.

Teacher cue: Use this for comparison, a second round, or a partner challenge.

### False Memory Word List Reveal / Twist

Student-facing stimulus 3: A recall sheet separating remembered words from inferred words. Before discussing, write what you noticed, what you inferred, and how confident you are.

Teacher cue: Teacher reveal: connect the changed response to the big idea — Coherence can produce confident false memory.


## Actual Lesson Questions

1. After inspecting False Memory Word List Stimulus A, what is your first knowledge claim?
   - Teacher key/cue: Teacher cue: look for a clear claim, not just a description.
2. Which exact detail from False Memory Word List Stimulus A did you use as evidence?
   - Teacher key/cue: Teacher cue: students should name visible words, data, source features, method features, or contextual clues.
3. What did you infer beyond the evidence?
   - Teacher key/cue: Teacher cue: this is where assumptions, framing, models, or perspective usually appear.
4. How confident are you: 50%, 60%, 70%, 80%, 90%, or 100%?
   - Teacher key/cue: Teacher cue: confidence is data for the debrief, not a grade.
5. What missing information would most improve the knowledge claim?
   - Teacher key/cue: Teacher cue: push for method, source, context, comparison, or definitions.
6. Does a coherent story make a belief feel more true?
   - Teacher key/cue: Teacher cue: students should move from the classroom example to a general TOK issue.

## Before the Activity

1. What do you think this activity will test about knowledge, evidence, or interpretation?
2. What assumptions might you bring to this activity?
3. How confident are you that your first judgement will be reliable? Explain briefly.

## Materials or Stimulus

- Classroom run kit with prompt cards, example stimuli, and debrief moves.
- PowerPoint deck with a student-facing sequence for running the activity.

## Activity Steps

1. Read the word list aloud once.
2. Students complete a distractor task for one minute.
3. Ask which words they remember. Include the lure word "sleep" on a recognition list.
4. Count how many students confidently remember the lure.

## Teacher Language

- Write your first judgement before we discuss it; the first answer is useful evidence.
- Separate what you noticed from what you inferred.
- What would make this knowledge claim more reliable?
