# Memory versus Monument Worksheet

Name: ________________________________  Class: __________________  Date: __________________

## Learning Focus

Public memory and historical knowledge overlap, but they are not the same practice.

## Student Prompt

Inspect Memory versus Monument Stimulus A. Record one first claim, the evidence you used, your confidence, and what would make you revise.

## Concrete Stimulus Packet

### Memory versus Monument Stimulus A

Student-facing stimulus 1: Fictional memorial: a plaque celebrating a former headteacher beside a diary extract from a student and a board meeting note. Mark what the source shows, what it omits, whose perspective appears, and what claim it can responsibly support.

Teacher cue: Students inspect this exact card first and commit to a claim before hearing anyone else.

### Memory versus Monument Stimulus B

Student-facing stimulus 2: Comparison prompt: What does each source ask the public to remember?. Mark what the source shows, what it omits, whose perspective appears, and what claim it can responsibly support.

Teacher cue: Use this for comparison, a second round, or a partner challenge.

### Memory versus Monument Reveal / Twist

Student-facing stimulus 3: Comparison prompt: What does each source ask the public to remember?. Mark what the source shows, what it omits, whose perspective appears, and what claim it can responsibly support.

Teacher cue: Teacher reveal: connect the changed response to the big idea — Public memory and historical knowledge overlap, but they are not the same practice.


## Actual Lesson Questions

1. After inspecting Memory versus Monument Stimulus A, what is your first knowledge claim?
   - Teacher key/cue: Teacher cue: look for a clear claim, not just a description.
2. Which exact detail from Memory versus Monument Stimulus A did you use as evidence?
   - Teacher key/cue: Teacher cue: students should name visible words, data, source features, method features, or contextual clues.
3. What did you infer beyond the evidence?
   - Teacher key/cue: Teacher cue: this is where assumptions, framing, models, or perspective usually appear.
4. How confident are you: 50%, 60%, 70%, 80%, 90%, or 100%?
   - Teacher key/cue: Teacher cue: confidence is data for the debrief, not a grade.
5. What missing information would most improve the knowledge claim?
   - Teacher key/cue: Teacher cue: push for method, source, context, comparison, or definitions.
6. How is public memory different from historical knowledge?
   - Teacher key/cue: Teacher cue: students should move from the classroom example to a general TOK issue.

## Activity Steps

1. Show a memorial image and ask what knowledge it seems to communicate.
2. Give a personal memory and an archival source about the same event.
3. Students compare purpose, evidence, emotion, and omissions.
4. Groups decide what each source helps us know and what it cannot settle.
5. Debrief the difference between remembering and explaining the past.

## Teacher Language

- A memorial is not a textbook page; it is a public act of selection.
- What kind of truth is the memorial trying to preserve?
